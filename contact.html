Frequently Asked Questions

Is PDFMerge really free?

Yes! PDFMerge is completely free to use. There are no hidden fees, no premium features, and no credit card required. I created this tool specifically for students who need free alternatives to expensive PDF software.

How do you make money if it's free?

I don't make money from PDFMerge. This is a passion project created to help students. I believe that educational tools should be accessible to everyone, regardless of their financial situation.

Can I use PDFMerge for commercial purposes?

Yes, you can use PDFMerge for any purpose - personal, educational, or commercial. The tool is free for everyone to use.

What file formats does PDFMerge support?

Currently, PDFMerge only supports PDF files. The tool is designed specifically for merging PDF documents, which is the most common need for students and professionals.

Is there a file size limit?

There's no artificial file size limit imposed by PDFMerge. However, very large files may be limited by your browser's memory capacity. For best results, try to keep individual files under 100MB.

Can I contribute to PDFMerge?

Absolutely! PDFMerge is open source and I welcome contributions from other developers. You can find the source code on GitHub and submit issues, feature requests, or pull requests.

Your Feedback Matters

As a student developer, I'm constantly learning and improving. Your feedback helps me understand what features students need most and how to make PDFMerge better for everyone. Whether it's a bug report, feature suggestion, or just a kind word, I appreciate hearing from users.
